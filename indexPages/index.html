WebServer Project

About the Project

Welcome to the WebServer project! This project aims to create a custom web server with various features, including handling HTTP requests, serving static files, and managing client connections efficiently. The server supports multiple HTTP methods and provides a robust platform for serving web content.

This project is a part of the 42 coding curriculum, focusing on low-level server development and network programming.
